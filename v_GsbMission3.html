GSB Mission 3 :





Présentaion :


Le cas d'utilisation pour cette mission était le suivant :
Description du cas
Nom : voir les rapports d’un visiteur
Acteur : la secrétaire
Scénario nominal
1. La secrétaire demande à voir les rapports d’un visiteur
2. Le système retourne la liste des visiteurs
3. La secrétaire sélectionne un visiteur et valide
4. Le système retourne la liste de tous les rapports du visiteur.
Extension
5. La secrétaire demande une exportation au fichier XML
6. Le système retourne la liste de tous les rapports di visiteur au format XML
J'ai donc réalisé une application Windows Form C# qui répond au cas d'utilisation. Ma mission est composée de 2 pages. La première qui est la gestion des visiteurs (ajout,modif,suppression) connecté en base de donnée.


Et la deuxième est une page d'affichage des rapports selon le visiteur choisit avec une sortie au format xml pour les rapports choisit.
